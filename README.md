# ZenQI – Quality Intelligence Ecosystem

A modular Quality Intelligence platform with frontend UI components for requirements intelligence and quality analytics.

## Overview

ZenQI provides a frontend capability layer for quality engineering operations. This repository contains standalone HTML dashboards built with Tailwind CSS for on-premise deployment.

**Current Scope:** Frontend UI only. Backend services are planned for future releases.

## Modules

### DeepSpeci – Requirements Intelligence
- User story analysis and decomposition
- Test case design interface
- Requirements documentation

### Insights360 – Quality Intelligence Dashboard
- Test execution analytics
- Predictive analysis reports
- Root cause analysis
- Test strategy planning

## Quick Start

**Clone and open:**
```bash
git clone https://github.com/AbhilashGiridharan/ZenQIDemo.git
cd ZenQIDemo
```

**Option 1:** Open `frontend/index.html` in your browser

**Option 2:** Run local server:
```bash
cd frontend
python3 -m http.server 8080
```
Then visit `http://localhost:8080`

## Technology

- HTML5, Tailwind CSS
- Static hosting ready
- Modular plugin architecture

## Deployment

Deploy `frontend/` directory to any static web server (Nginx, Apache, S3, etc.)
